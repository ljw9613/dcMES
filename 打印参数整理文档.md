# 打印参数整理文档

## 概述
本文档整理了扫码转化页面中所有可用的打印参数，包括来源于不同数据库表的字段以及动态计算的字段。

## 数据来源

### 1. 基础条码数据
| 参数名 | 描述 | 来源 | 备注 |
|--------|------|------|------|
| `printBarcode` | 打印条码 | 扫描条码/预生产条码表 | 根据转化模式可能使用转化后的条码 |
| `printBarcodeText` | 条码文本 | 扫描条码/预生产条码表 | 与printBarcode对应的文本格式 |

### 2. 销售订单客户信息表 (k3_SAL_SaleOrder_CustInfo)

#### 基础订单信息
| 参数名 | 描述 | 数据类型 | 备注 |
|--------|------|----------|------|
| `FSaleOrderId` | 关联销售订单ID | String | 必填，关联主表 |
| `FBillNo` | 销售订单编号 | String | 必填 |
| `FCustPO` | 客户PO号 | String | 用于生成配网码 |
| `FCustPOLineNo` | 客户PO行号 | String | - |
| `FSapId` | SAP ID | String | - |
| `FPurchaseOrderNo` | 采购订单号 | String | - |

#### 客户信息
| 参数名 | 描述 | 数据类型 | 备注 |
|--------|------|----------|------|
| `FCustId` | 客户ID | String | 必填 |
| `FCustCode` | 客户编码 | String | 必填 |
| `FCustName` | 客户名称 | String | 必填 |

#### 客户物料信息
| 参数名 | 描述 | 数据类型 | 备注 |
|--------|------|----------|------|
| `FCustMaterialName` | 客户物料名称 | String | 中文名称 |
| `FCustMaterialNameEn` | 客户物料名称（英文） | String | 英文名称 |

#### 系统字段
| 参数名 | 描述 | 数据类型 | 备注 |
|--------|------|----------|------|
| `FCreateTime` | 创建时间 | Date | 默认当前时间 |
| `FModifyTime` | 修改时间 | Date | - |
| `FRemark` | 备注 | String | - |
| `FStatus` | 状态 | String | 默认'ENABLE' |

### 4. 动态计算字段

#### 网址相关
| 参数名 | 描述 | 计算逻辑 | 备注 |
|--------|------|----------|------|
| `FProductUrl` | 产品网址（含序列号） | FProductUrl + "&sn=" + printBarcode | 仅当原始FProductUrl存在时 |

#### 配网码
| 参数名 | 描述 | 计算逻辑 | 备注 |
|--------|------|----------|------|
| `FNetCode` | 配网码 | generateCompleteNetCode(FCustPO, printBarcode) | 使用加密算法生成 |

#### 日期相关
| 参数名 | 描述 | 计算逻辑 | 备注 |
|--------|------|----------|------|
| `ProductionDate` | 生产日期 | MM/YYYY格式 | 当前月份/年份 |
| `createDate` | 创建日期 | YYYY.MM格式 | 当前年份.月份 |
| `wxxcqProductionDate` | 无线吸尘器生产日期 | YYYY.MM格式 | 当前年份.月份 |

#### 综合字段
| 参数名 | 描述 | 计算逻辑 | 备注 |
|--------|------|----------|------|
| `printBarcodeZH` | 追觅综合字段 | printBarcode + "," + FCustPO + "," + productionDate + "," + quantity | 逗号分隔的组合字段 |
| `printBarcodeZH2025` | 追觅综合字段固定2025字段 | printBarcode + "," + FCustPO + "," + "20251221" + "," + quantity | 逗号分隔的组合字段 |
